# MercadoShop

MercadoShop es una plataforma de marketplace que conecta a tiendas/vendedores con clientes, proporcionando un panel de administración robusto para cada tipo de usuario.

## Características
- **Multiperfil:** Soporte para Super Administradores, Trabajadores (Vendedores) y Clientes.
- **Sidebars Unificadas:** Interfaz consistente y estandarizada en todos los paneles de control mediante Tailwind CSS.
- **Gestión de Pedidos y Catálogo:** Flujos completos para añadir productos y hacer seguimiento de compras.
- **Diseño Responsivo:** UI optimizada utilizando los estándares de Tailwind.
